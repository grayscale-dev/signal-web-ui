# signal-textarea

Multiline text input with auto-resize option.

## Usage
```html
<signal-textarea rows="4" placeholder="Enter notes"></signal-textarea>
```

## Props
- `value` string
- `defaultValue` string
- `rows` number
- `autoResize` boolean
- `maxLength` number
- `placeholder` string
- `disabled` boolean
- `validationState` string
