# signal-web-ui repo export

This directory contains everything needed to run the Signal UI monorepo.

## Included packages

- `packages/core`
- `packages/design-tokens`
- `packages/tsconfig`
- `packages/eslint-config`
- `packages/react`
- `packages/vue`
- `packages/angular`
- `packages/ember`

## Getting started

```bash
pnpm install
pnpm build
pnpm test
```
